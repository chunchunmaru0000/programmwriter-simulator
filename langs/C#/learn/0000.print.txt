Функция print в Python используется для вывода информации на экран.
 Вы можете передать print любое количество объектов (например, строки,
 числа и переменные), и они будут отображены в консоли или другом месте,
 куда направлен вывод. Вот простой пример:

print("Привет, мир!")

Этот код выведет текст "Привет, мир!" в консоли. Также можно
